Project status

RockScript does not offer any stability guarantees at this point.

RockScript is early stage and the implementation is very volatile.

At this point we have a good start on the developer experience. You -as a curious, badass, rock star developer- can already try it as intended to get a good idea of what RockScript can do. You can help us with your feedback.
Let us know what you think of the RockScript concepts and goals. Any question, suggestion or other other form of feedback is really appreciated.

Persistence is still only in-memory. We plan to build persistence to a datastore (SQL or Kafka) once the developer experience is sufficiently done.

Also at this stage the coverage of the JavaScript language is still minimal.

We offer commercial support and a product on top of the open source engine to fund further development of the project.
